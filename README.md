AI-Powered Email Assistant

Built an end-to-end AI automation system using Google's Gemini API that connects to a Gmail account (tested via a sandboxed test account to ensure safe experimentation) to fetch, summarize, and prioritize emails. Implemented customizable auto-reply generation with real email dispatch capability, verified end-to-end by confirming delivery in a live inbox. Compared a fine-tuned model approach against the API-based approach to evaluate accuracy and performance.
